# Grammatica Italiana

## Project Overview

This is a React application built with TypeScript and Vite, designed to help students learn Italian grammar from English.

Interactive charts for understanding:
*   **Articles:** Definite and Indefinite articles.
*   **Verbs:** Present Indicative conjugation for -ARE, -ERE, and -IRE verbs.
*   **Prepositions:** Simple and Articulated prepositions.
*   **Vocabulary:** Categorized word lists with gender and English translations.

## Tech Stack
*   React 19
*   TypeScript
*   Vite
*   Tailwind CSS v4
*   Framer Motion
*   Lucide React
